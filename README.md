# 체크인오늘 (CheckinToday)

오늘을 체크인 🌿

ADHD·완벽주의 성향의 미루니들을 위한, 압박 없이 쓸 수 있는 할 일 관리 앱

## 시작하기

```bash
npm install
npm run dev
```

## 배포하기

```bash
npm run build
```

`dist` 폴더가 생성되면 Vercel, Firebase Hosting 등에 배포할 수 있습니다.
